# Burnout Pending

An exciting journey into the world of development: help a game dev stay in a state of flow.

## How to play

Go to the armchair to enter a flow state. Collect lamps, avoid poop. Then go to bed, this will start a new day.

## Controls

* Move: WASD, Arrow, Gamepad, Mouse, Touchscreen
* Pause: Esc, Q, Z, P, Back, Left bumper
* Action: Space, C, Mouse right click, Start, Right bumper
* Zoom-in: Equal, Mouse wheel down, Right action
* Zoom-out: Minus, Mouse wheel up, Left action
